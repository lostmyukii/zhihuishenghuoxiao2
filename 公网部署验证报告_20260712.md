## 5. 实板终验步骤

服务器链路已完成。实板 Web Serial 终验需要在插着 N16R8 的电脑上进行：

1. 用 Chrome 或 Edge 打开 `https://hongkongxiao2.ilelezhan.cn/`。
2. 点击“选择串口”，选择 CH340 对应串口。
3. 确认顶部依次显示 `USB 已连接`、`WSS 在线`、`MQTT 在线`、`开发板在线`。
4. 在第二台电脑或手机打开同一网址，确认 8 路传感器数据同步变化。
5. 在第二台设备切换模式，确认插板电脑日志显示“云端命令已写入 USB”，并收到开发板真实 `ack`。

只有完成第 3 至第 5 步，才算完成真实 USB 到公网双设备终验。WSS/MQTT 在线本身不等于开发板在线。

## 6. 安全说明

- 部署没有保存或提交 SSH 密码。
- MQTT 只监听 `127.0.0.1`，没有向公网开放匿名端口。
- Nginx 每次变更前均先通过 `nginx -t`，只执行平滑 reload，没有 restart。
- 继电器继续只允许连接低压演示灯，不接 `220V`。
